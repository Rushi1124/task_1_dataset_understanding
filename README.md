# Task 1: Understanding Dataset & Data Types

## Objective
To analyze and understand the structure, data types, and machine learning readiness of a dataset.

## Dataset
Titanic Dataset (publicly available)

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Work Done
- Loaded the dataset and inspected first and last records
- Identified numerical, categorical, ordinal, and binary features
- Analyzed data types, missing values, and statistical summaries
- Identified target variable and class imbalance
- Added a dataset analysis report in the notebook

## Outcome 
Developed a clear understanding of dataset structure, quality, and suitability for machine learning.
